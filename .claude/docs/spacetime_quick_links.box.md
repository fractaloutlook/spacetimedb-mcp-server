## Useful Endpoints
- `/v1/database/:name_or_identity/schema` - Get database schema
- `/v1/database/:name_or_identity/sql` - Execute SQL queries (POST)
- `/v1/database/:name_or_identity/call/:reducer` - Call a reducer (POST)
- `/v1/database/:name_or_identity/subscribe` - WebSocket subscription (GET)
